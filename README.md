## 🚀 Overview

**AI Dev Team** is a multi-agent autonomous software engineering system that transforms a **product idea into a fully working application**.

Unlike traditional AI tools that generate code in a single step, this system simulates a **real software engineering team**, where each agent performs a specialized role in the development lifecycle.

---

## 🧠 Core Idea

Traditional approach:
<!-- <img width="681" height="130" alt="image" src="https://github.com/user-attachments/assets/4bc76789-26d6-46f2-ac2b-8844a83a2fec" /> -->
Prompt → Code
<br/><br/>
AI Dev Team approach<br/>
Requirement → Specification → Architecture → Planning → Coding → Review → Testing → Deployment



Each stage is executed by a **dedicated AI agent** inside a **graph-based workflow**.

---

## 🏗️ System Architecture

The system models real-world engineering teams using specialized agents:

- 🧑‍💼 Product Manager  
- 🏗️ Software Architect  
- 🧠 Planner  
- 💻 Developer  
- 🔍 Code Reviewer  
- 🐞 Debugger  
- 🚀 Deployment Engineer  

Each agent operates as a **node in a workflow graph**, ensuring structured and modular execution.

---

## ⚙️ Tech Stack

- **Workflow Engine:** LangGraph  
- **LLM Engine:** Gemini  
- **Runtime:** Node.js  
- **Interface:** CLI + Web App  
- **Deployment:** Vercel

---

## 🔄 Execution Flow

1. Requirement is received  
2. Product Manager Agent analyzes the requirement  
3. If ambiguity is detected:
   - Clarifying questions are generated  
4. User provides answers  
5. PM Agent generates structured specification  
6. System proceeds through all agents step-by-step  
7. Final application is produced
